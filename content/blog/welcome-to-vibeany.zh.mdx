# 欢迎使用 VibeAny (´∀｀)

欢迎来到 **VibeAny**，一个专为现代开发者精心打造的 AI 应用开发框架。基于 Next.js 15、TypeScript 和前沿 Web 技术，让您的 AI 想法轻松变为现实。

## 什么是 VibeAny？ (・∀・)

VibeAny 是一个企业级的全栈启动套件，专为构建高质量 AI 应用而生。我们为您准备了：

- **安全认证体系** (￣▽￣) - 支持邮箱密码、第三方登录和无密码登录
- **智能 AI 集成** (◕‿◕) - 预配置主流 AI 服务，开箱即用
- **灵活支付方案** (＾◡＾) - 完整的 Stripe 集成，订阅制和单次付费随心选
- **全球化支持** (☆▽☆) - 内置多语言切换，触达全球用户
- **现代化界面** (◠‿◠) - 自适应设计，深色模式，无障碍体验

## 三分钟上手指南 (ﾟ∀ﾟ)

跟随这些简单步骤，让您的 AI 应用快速启动：

```bash
# 获取项目源码
git clone https://github.com/jiahao-jayden/vibe-any

# 安装项目依赖
bun i

# 配置环境变量
cp .env.example .env

# 启动开发服务器
bun dev
```
