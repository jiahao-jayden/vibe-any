## 概览

本指南介绍 `src/config/` 目录内的配置文件，每个文件负责网站的某一部分。带有（需改）的通常需要你根据项目实际修改；未标注的以了解为主。

```bash
src/config/
  core/website-config.tsx      # 网站核心设置（元信息、i18n、监控、支付、价格计划）
  layout/header-config.tsx     # 顶部导航与 Logo
  layout/footer-config.tsx     # 页脚分区与社交链接
  pages/pricing/pricing-config.tsx # 价格页文案（从网站配置读取计划）
  features/payment-config.tsx  # 价格计划辅助（翻译等）
```

<Callout>
所有配置均具备类型约束；除特别说明外，可在服务端与客户端使用。
</Callout>
