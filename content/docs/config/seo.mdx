title: SEO
description: Localized SEO configuration under src/config/i18n/seo
---

SEO is configured per-locale in `src/config/i18n/seo/` and selected via `getSeoConfig(locale)`.

### Files

- `seo-config.tsx`: shared types and `getSeoConfig`
- `seo-en-config.tsx`: English SEO
- `seo-zh-config.tsx`: Chinese SEO

### Fields and examples

#### Basics

| Field | Type | Purpose | Example |
| --- | --- | --- | --- |
| `title.default` | `string` | Default page title | `"VibeAny - Your AI Copilot"` |
| `title.template` | `string` | Title template (`%s` is page title) | `"%s | VibeAny"` |
| `description` | `string` | Default meta description | `"All-in-one AI integration platform"` |
| `keywords` | `string[]` | Keywords | `["AI","SaaS","VibeAny"]` |

#### Alternates

| Field | Type | Purpose | Example |
| --- | --- | --- | --- |
| `alternates.canonical` | `string` | Canonical URL | `"https://vibeany.com/en"` |
| `alternates.languages` | `Record<string,string>` | Language URLs | `{ "en-US": "https://vibeany.com/en", "zh-CN": "https://vibeany.com/zh" }` |

#### Open Graph

| Field | Type | Purpose | Example |
| --- | --- | --- | --- |
| `openGraph.title` | `string` | OG title | `"VibeAny - Your AI Copilot"` |
| `openGraph.description` | `string` | OG description | `"Integrate top-tier models"` |
| `openGraph.url` | `string` | Page URL | `"https://vibeany.com/en"` |
| `openGraph.siteName` | `string` | Site name | `"VibeAny"` |
| `openGraph.images[].url` | `string` | Image URL | `"/og.avif"` |
| `openGraph.images[].width` | `number` | Width | `1200` |
| `openGraph.images[].height` | `number` | Height | `630` |
| `openGraph.images[].alt` | `string` | Alt text | `"VibeAny Platform"` |
| `openGraph.locale` | `string` | Locale | `"en_US"` |
| `openGraph.type` | `string` | Type | `"website"` |

#### Twitter Card

| Field | Type | Purpose | Example |
| --- | --- | --- | --- |
| `twitter.card` | `'summary_large_image' \| 'summary'` | Card type | `'summary_large_image'` |
| `twitter.title` | `string` | Title | `"VibeAny - Your AI Copilot"` |
| `twitter.description` | `string` | Description | `"All-in-one AI integration platform"` |
| `twitter.creator` | `string` | Creator handle | `'@vibeany'` |
| `twitter.images[]` | `string[]` | Image URLs | `["/twitter-image.avif"]` |

#### Full example

```ts
export const seo = {
  title: { default: 'VibeAny - Your AI Copilot', template: '%s | VibeAny' },
  description: 'All-in-one AI integration platform',
  keywords: ['AI','SaaS','VibeAny'],
  alternates: {
    canonical: 'https://vibeany.com/en',
    languages: { 'en-US': 'https://vibeany.com/en', 'zh-CN': 'https://vibeany.com/zh' },
  },
  openGraph: {
    title: 'VibeAny - Your AI Copilot',
    description: 'Integrate top-tier models',
    url: 'https://vibeany.com/en',
    siteName: 'VibeAny',
    images: [{ url: '/og.avif', width: 1200, height: 630, alt: 'VibeAny Platform' }],
    locale: 'en_US',
    type: 'website',
  },
  twitter: {
    card: 'summary_large_image',
    title: 'VibeAny - Your AI Copilot',
    description: 'All-in-one AI integration platform',
    creator: '@vibeany',
    images: ['/twitter-image.avif'],
  },
}
```

> Use assets under `/public` or a stable CDN for images.
