title: Footer Menus
description: Configure footer sections and social links in src/config/layout/footer-config.tsx
---

Footer content comes from `getFooterConfig(locale)` and `getFooterSocialLinks()`.

## Sections

Each section has a `title` and an array of `{ title, href }` links.

## Social links

`getFooterSocialLinks()` returns a list of links with icons. Update the array to add/remove items.

### Example

```ts
export function getFooterSocialLinks() {
  return [
    {
      href: '/',
      ariaLabel: 'Email',
      icon: <Mail className='h-5 w-5' strokeWidth={1.5} />,
    },
    {
      href: '/',
      ariaLabel: 'YouTube',
      icon: <Youtube className='h-5 w-5' />,
    },
  ]
}
```
