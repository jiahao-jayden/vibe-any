---
title: 头部（Header）
description: 在 src/config/layout/header-config.tsx 配置站点 Logo 与导航
---

`src/config/layout/header-config.tsx` 会根据语言返回头部配置。

## 可配置项

- `logo.title`、`logo.image.src`、`logo.image.alt`
- `nav.items`: `{ title, href }[]`

## 示例

```ts
export async function getHeaderConfig(locale: string) {
  return {
    logo: {
      title: 'VibeAny',
      image: {
        src: '/logo.avif',
        alt: 'Logo',
      },
    },
    nav: {
      items: [
        {
          title: '文档',
          href: '/docs',
        },
      ],
    },
  }
}
```
