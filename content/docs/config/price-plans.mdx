---
title: Price Plans
description: Plans and prices defined in website config (Stripe price IDs)
---

Plans live in `websiteConfig.planWithPrice`. For localized display use `getPricePlans(locale)`.

### Required env variables

| Plan | Env |
| ---- | --- |
| Pro Monthly | `NEXT_PUBLIC_STRIPE_PRO_MONTHLY_PRICE_ID` |
| Pro Yearly | `NEXT_PUBLIC_STRIPE_PRO_YEARLY_PRICE_ID` |
| Lifetime | `NEXT_PUBLIC_STRIPE_LIFETIME_PRICE_ID` |

Add or remove plans by editing `planWithPrice`.

### Example structure

```ts
planWithPrice: [
  {
    id: 'pro',
    planType: 'subscription',
    credit: {
      amount: 100,
      expireDays: 31,
    },
    prices: [
      {
        type: 'subscription',
        priceId: process.env.NEXT_PUBLIC_STRIPE_PRO_MONTHLY_PRICE_ID!,
        amount: 990,
        currency: 'USD',
        interval: 'month',
      },
    ],
  },
]
```
