---
title: Header
description: Configure the header logo and navigation in src/config/layout/header-config.tsx
---

`src/config/layout/header-config.tsx` returns the header configuration based on locale.

## What you can change

- `logo.title`, `logo.image.src`, `logo.image.alt`
- `nav.items`: `{ title, href }[]`

## Example

```ts
export async function getHeaderConfig(locale: string) {
  return {
    logo: {
      title: 'VibeAny',
      image: {
        src: '/logo.avif',
        alt: 'Logo',
      },
    },
    nav: {
      items: [
        {
          title: 'Docs',
          href: '/docs',
        },
      ],
    },
  }
}
```
