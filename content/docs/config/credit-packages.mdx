---
title: Credit Packages
description: Optional one-time credit bundles under payment.credit.packages
---

Define credit packages in `websiteConfig.payment.credit.packages`.

Each package requires:

- `id`: unique string
- `credit.amount`, `credit.expireDays`
- `price.priceId`, `price.amount`, `price.currency`

Use helpers in `config/features/payment-config.tsx` to transform packages into displayable plans for the UI.

### Example

```ts
payment: {
  credit: {
    packages: [
      {
        id: 'credit_100',
        credit: {
          amount: 100,
          expireDays: 31,
        },
        price: {
          priceId: process.env.NEXT_PUBLIC_STRIPE_100_CREDITS_PRICE_ID!,
          amount: 990,
          currency: 'USD',
        },
      },
    ],
  },
}
```
