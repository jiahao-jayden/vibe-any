页脚内容来源于 `getFooterConfig(locale)` 与 `getFooterSocialLinks()`。

## 分区（Sections）

每个分区包含 `title` 与若干 `{ title, href }` 链接。

## 社交链接（Social Links）

`getFooterSocialLinks()` 返回包含图标的链接数组，直接增删数组项即可。

### 示例

```ts
export function getFooterSocialLinks() {
  return [
    {
      href: '/',
      ariaLabel: '邮箱',
      icon: <Mail className='h-5 w-5' strokeWidth={1.5} />,
    },
    {
      href: '/',
      ariaLabel: 'YouTube',
      icon: <Youtube className='h-5 w-5' />,
    },
  ]
}
```
