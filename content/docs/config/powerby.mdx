---
title: Powered by Logos
description: Configure partner/brand logos shown on the landing page
---

Logos are defined in `src/config/pages/landing/powerby-config.tsx` and consumed by the landing page.

## File

`src/config/pages/landing/powerby-config.tsx`

```ts
export type PowerByItem = {
  name: string
  path: string
}

export function getPowerByConfig(): PowerByItem[] {
  return [
    { name: 'github', path: '/partners/github.svg' },
    { name: 'tailwindcss', path: '/partners/tailwindcss.svg' },
  ]
}
```

## What you can change

- Add/remove items in the returned array.
- `name`: identifier (used for alt/analytics).
- `path`: image path under `/public/partners` (SVG/AVIF/PNG all fine).

## Example (multi-line)

```ts
export function getPowerByConfig(): PowerByItem[] {
  return [
    {
      name: 'github',
      path: '/partners/github.svg',
    },
    {
      name: 'vercel',
      path: '/partners/vercel.svg',
    },
  ]
}
```
