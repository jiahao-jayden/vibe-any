---
title: 合作/品牌 Logo
description: 配置落地页展示的合作/品牌 Logo
---

Logo 列表定义在 `src/config/pages/landing/powerby-config.tsx`，由落地页读取并渲染。

## 文件

`src/config/pages/landing/powerby-config.tsx`

```ts
export type PowerByItem = {
  name: string
  path: string
}

export function getPowerByConfig(): PowerByItem[] {
  return [
    { name: 'github', path: '/partners/github.svg' },
    { name: 'tailwindcss', path: '/partners/tailwindcss.svg' },
  ]
}
```

## 可修改内容

- 在返回数组中增删条目
- `name`：标识（可用于 alt/埋点）
- `path`：图片路径，位于 `/public/partners`（SVG/AVIF/PNG 均可）

## 多行示例

```ts
export function getPowerByConfig(): PowerByItem[] {
  return [
    {
      name: 'github',
      path: '/partners/github.svg',
    },
    {
      name: 'vercel',
      path: '/partners/vercel.svg',
    },
  ]
}
```
