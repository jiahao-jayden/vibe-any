---
title: 积分套餐
description: 在 payment.credit.packages 下定义一次性积分包
---

在 `websiteConfig.payment.credit.packages` 下定义积分套餐。

每个套餐包含：

- `id`：唯一标识
- `credit.amount`、`credit.expireDays`
- `price.priceId`、`price.amount`、`price.currency`

可使用 `config/features/payment-config.tsx` 的辅助方法将套餐转换为前端可展示的计划。

### 示例

```ts
payment: {
  credit: {
    packages: [
      {
        id: 'credit_100',
        credit: {
          amount: 100,
          expireDays: 31,
        },
        price: {
          priceId: process.env.NEXT_PUBLIC_STRIPE_100_CREDITS_PRICE_ID!,
          amount: 990,
          currency: 'USD',
        },
      },
    ],
  },
}
```
