SEO 按语言分别配置在 `src/config/i18n/seo/`，通过 `getSeoConfig(locale)` 选择。

### 文件

- `seo-config.tsx`：共享类型与 `getSeoConfig`
- `seo-en-config.tsx`：英文 SEO
- `seo-zh-config.tsx`：中文 SEO

### 字段说明与示例

#### 基本信息

| 字段 | 类型 | 作用 | 示例 |
| --- | --- | --- | --- |
| `title.default` | `string` | 页面默认标题 | `"VibeAny - 您的 AI 助手"` |
| `title.template` | `string` | 带占位的标题模板（`%s` 为页面标题） | `"%s | VibeAny"` |
| `description` | `string` | 站点默认描述（用于 `<meta name="description">`） | `"一体化 AI 集成平台"` |
| `keywords` | `string[]` | 关键词数组 | `["AI","SaaS","VibeAny"]` |

#### 备用链接（alternates）

| 字段 | 类型 | 作用 | 示例 |
| --- | --- | --- | --- |
| `alternates.canonical` | `string` | 规范化链接 | `"https://vibeany.com/zh"` |
| `alternates.languages` | `Record<string,string>` | 多语言页面地址 | `{ "en-US": "https://vibeany.com/en", "zh-CN": "https://vibeany.com/zh" }` |

#### Open Graph（社交分享）

| 字段 | 类型 | 作用 | 示例 |
| --- | --- | --- | --- |
| `openGraph.title` | `string` | OG 标题 | `"VibeAny - 您的终极 AI 助手"` |
| `openGraph.description` | `string` | OG 描述 | `"集成顶级模型，效率倍增"` |
| `openGraph.url` | `string` | 页面 URL | `"https://vibeany.com/zh"` |
| `openGraph.siteName` | `string` | 站点名称 | `"VibeAny"` |
| `openGraph.images[].url` | `string` | 图片地址（支持 `/public` 或 CDN） | `"/og.avif"` |
| `openGraph.images[].width` | `number` | 图片宽度 | `1200` |
| `openGraph.images[].height` | `number` | 图片高度 | `630` |
| `openGraph.images[].alt` | `string` | 图片替代文本 | `"VibeAny 平台"` |
| `openGraph.locale` | `string` | 语言区域 | `"zh_CN"` |
| `openGraph.type` | `string` | 类型 | `"website"` |

#### Twitter Card

| 字段 | 类型 | 作用 | 示例 |
| --- | --- | --- | --- |
| `twitter.card` | `'summary_large_image' \| 'summary'` | 卡片类型 | `'summary_large_image'` |
| `twitter.title` | `string` | 标题 | `"VibeAny - 您的终极 AI 助手"` |
| `twitter.description` | `string` | 描述 | `"一体化 AI 集成平台"` |
| `twitter.creator` | `string` | 作者/账号 | `'@vibeany'` |
| `twitter.images[]` | `string[]` | 图片地址数组 | `["/twitter-image.avif"]` |

#### 完整示例

```ts
export const seo = {
  title: { default: 'VibeAny - 您的终极 AI 助手', template: '%s | VibeAny' },
  description: '一体化 AI 集成平台',
  keywords: ['AI','SaaS','VibeAny'],
  alternates: {
    canonical: 'https://vibeany.com/zh',
    languages: { 'en-US': 'https://vibeany.com/en', 'zh-CN': 'https://vibeany.com/zh' },
  },
  openGraph: {
    title: 'VibeAny - 您的终极 AI 助手',
    description: '集成顶级模型，效率倍增',
    url: 'https://vibeany.com/zh',
    siteName: 'VibeAny',
    images: [{ url: '/og.avif', width: 1200, height: 630, alt: 'VibeAny 平台' }],
    locale: 'zh_CN',
    type: 'website',
  },
  twitter: {
    card: 'summary_large_image',
    title: 'VibeAny - 您的终极 AI 助手',
    description: '一体化 AI 集成平台',
    creator: '@vibeany',
    images: ['/twitter-image.avif'],
  },
}
```

> 图片建议放置在 `/public` 下，或使用稳定的 CDN 链接。
