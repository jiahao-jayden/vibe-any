## Overview

This guide explains the configuration files under `src/config/`. Each file controls a specific area of the site. Items marked with (edit) typically need changes; others are informational.

```bash
src/config/
  core/website-config.tsx      # Core site settings (metadata, i18n, monitoring, payment, plans)
  layout/header-config.tsx     # Navbar menus & logo
  layout/footer-config.tsx     # Footer sections & social links
  pages/pricing/pricing-config.tsx # Pricing UI texts (reads plans from website config)
  features/payment-config.tsx  # Helpers for price plans (translations)
```

<Callout>
All configuration is type-safe and can be imported both on the server and client unless noted otherwise.
</Callout>
