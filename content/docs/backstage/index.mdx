# 管理后台

boost-any 提供了功能完整的管理后台，让你可以轻松管理用户、内容和系统配置。

## 🔑 访问管理后台

### 管理员权限

只有在环境变量 `ADMIN_EMAILS` 中配置的邮箱地址才能访问管理后台：

```bash title=".env"
ADMIN_EMAILS="admin@example.com,manager@example.com"
```

### 访问路径

管理后台访问地址：`/admin`

## 📊 功能模块

### 用户管理

- 用户列表查看
- 用户信息编辑
- 用户状态管理
- 权限分配

### 内容管理

- 页面内容编辑
- 文档管理
- 媒体文件管理
- SEO 配置

### 数据统计

- 用户增长统计
- 使用情况分析
- 收入报表
- 性能监控

### 系统配置

- 站点设置
- 功能开关
- 主题配置
- 邮件模板

## 🛡️ 安全特性

- 基于邮箱的权限控制
- 操作日志记录
- 敏感操作确认
- 自动登出机制

## 🚀 快速开始

1. 配置管理员邮箱
2. 登录系统
3. 访问 `/admin` 路径
4. 开始管理
