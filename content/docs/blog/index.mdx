# Blog

MDX-powered blog with categories, authors and i18n. The listing page is `/blog` and supports category filtering and SEO-friendly ordering.

<Callout type="info">
Posts live in `content/blog`. Use locale suffixes like `post.zh.mdx` for Chinese.
</Callout>

## Blog System Structure

- Content: `content/blog`, `content/author`, `content/category`
- App routes: `src/app/[locale]/(site)/blog` (listing) and `src/app/[locale]/(site)/blog/[...slug]` (post)
- Components: `src/components/blog` (`blog-grid.tsx`, `blog-card.tsx`, filters, etc.)
- Utils: `src/lib/blog/get-categories.ts`

## Source Configuration

This project uses Content Collections. The schemas are defined in `content-collections.ts`:

```ts title="content-collections.ts (excerpt)"
// Posts collection
const posts = defineCollection({
  name: "post",
  directory: "content/blog",
  include: "**/*.mdx",
  schema: z.object({
    title: z.string(),
    description: z.string(),
    image: z.string().optional(),
    date: z.string().datetime(),
    published: z.boolean().default(true),
    categories: z.array(z.string()),
    author: z.string(),
    tags: z.array(z.string()).optional(),
    estimatedTime: z.number().optional(),
  }),
})

// Authors & categories collections are also defined similarly
```

The listing imports generated data from `.content-collections/generated` and applies locale/category filters:

```tsx title="src/app/[locale]/(site)/blog/page.tsx (excerpt)"
import { allPosts } from "../../../../../.content-collections/generated"

const filteredPosts = allPosts
  .filter((post) => post.locale === locale && /* optional cat filter */ true)
  .sort((a, b) => toTime(b.date) - toTime(a.date))
```

## Multi-language Support

Create `filename.mdx` for the default locale and `filename.zh.mdx` for Chinese. Authors and categories follow the same pattern.

## Best Practices

- Always set `date` and `published` for predictable ordering
- Provide an `image` for richer cards and social previews
![](/docs/blog/blog-card.avif)
- Keep titles concise; use `description` for context
- Prefer localized files for fully translated experiences

## Programmatic Queries (optional)

```ts title="Filter posts by category"
import { allPosts } from "@/.content-collections/generated"

export const getPostsByCategory = (slug: string, locale: string) =>
  allPosts.filter((p) => p.locale === locale && p.categories?.some((c: any) => c?.slug === slug))
```
